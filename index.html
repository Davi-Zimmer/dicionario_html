Dicionario html

Tag img

<img/ src="Link" alt="Mensagem de carregamento">

Essa tag te permite botar imagens no site com o link ou caminho da imagem.

Da pra botar um texto pra caso a imagem não carregue.

Tag Video

<video src="Link" controls></video>

Essa tag te permite botar videos da mesma forma que a tag imagem.

"Control" mostra o display de controles, play, pause e a timeline.

Tag de paragráfo

<p>Meu parágrafo</p>

Não tem muito o que dizer sobre essa tag, ja que é só um paragrafo.

Tag br

<b/>

Da um pequeno espaço no documento.

Tag a

<a href="Link">Click me</a>

Essa tag te direciona pra outro site quando clicado.

Tag script

<script src="Link pro script">
// meu código
</script>

Essa tag te permite programar em js dentro de si.

Tag span

<span>conteúdo<span>

Essa tag é similar à tag p, mas é usada em lugares que não são tão importantes de fato.

Tag button

<button onclick="função" >mensagem do botão<button>

A tag button é usada pra ativar funções dos scripts com eventos do mouse.

Tag canvas

<canvas><canvas>

Essa tag permite criar um local de desenho no documento.

usado pra criar alguns jogos e renderizar coisas em 3d.

Tag style

<style>código CSS<style>

Essa tag pertite dentro de si a estilização de todas as outras tags usando de códigos CSS.
